App Template Operations

New File

Placeholder only, does nothing useful.

Open File

Opens a file and displays it in the window. Scroll bar works to see the whole file.

Save File

Displays a Save File Dialog and saves the file to the file named.

Print

Will do all the normal things to print a file.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.

Menu Tools/Test

Test

Executes a command which outputs "Hello World" to the display. This can be output using the Save as Command.
